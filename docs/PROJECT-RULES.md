# Project Rules

## General

Keep everything simple.

Avoid unnecessary complexity.

The public website drives the design of the admin portal.

Never change the database schema without discussion.

---

## Development

Use HTML, CSS and Vanilla JavaScript.

Prefer readability over clever code.

Reuse existing code where possible.

Do not introduce frameworks.

---

## UI

Maintain a consistent design.

Keep pages responsive.

Accessibility and SEO should never be broken.

---

## Vehicle Data

Never invent new fields.

If a new field is required, discuss it before implementation.

---

## Images

Cover Image is selected from uploaded Exterior images.

Exterior and Interior images are stored separately.

Poster is optional.

Auction Sheet is optional.

---

## AI Studio

Use AI Studio for UI generation.

Use ChatGPT for architecture, review and planning.

---

## Our Principles

1. Keep the application simple.
2. Keep the development process simple.
3. Build only what helps sell cars.
4. Don't over-engineer.
